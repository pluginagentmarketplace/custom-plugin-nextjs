# /agents - Find Your Specialist Agent

Discover which of the 7 specialist agents can help you.

## Available Agents

1. **Frontend & UI Developer** - HTML, CSS, JavaScript, React, Vue, Angular, design systems
2. **Backend & Server Developer** - Node.js, PHP, Java, Laravel, Spring Boot, APIs
3. **Mobile Developer** - iOS, Android, React Native, Flutter
4. **Data & AI Engineer** - Python, ML, AI, LLMs, MLOps
5. **DevOps & Cloud Engineer** - AWS, Kubernetes, Docker, Terraform, CI/CD
6. **System Architect** - System design, algorithms, architecture patterns
7. **Emerging Tech & Specialist** - Blockchain, security, QA, specialized roles

## Usage

```
/agents
/agents frontend
/agents backend
/agents data
/agents cloud
```

## Agent Selection

Based on your interest:
- Learning web? → Frontend or Backend agent
- Building mobile apps? → Mobile agent
- Working with data/AI? → Data & AI agent
- Managing infrastructure? → DevOps & Cloud agent
- Designing systems? → System Architect agent
- Exploring new tech? → Emerging Tech agent
