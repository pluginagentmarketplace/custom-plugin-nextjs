# /learn - Start Your Learning Journey

Build a personalized learning path based on your goals and experience level.

## Usage

```
/learn
/learn "become a frontend developer"
/learn "learn python"
/learn full-stack
```

## Learning Path Builder

When you use `/learn`, I'll ask you:

1. **What's your goal?**
   - Learn a specific technology
   - Become a developer in an area
   - Transition to a new role
   - Deepen existing skills

2. **What's your current level?**
   - Absolute beginner
   - Some experience
   - Intermediate
   - Advanced

3. **How much time per week?**
   - 5-10 hours
   - 10-20 hours
   - 20+ hours

4. **What's your learning style?**
   - Video courses
   - Reading & coding
   - Interactive tutorials
   - Mix of all

## Typical Learning Paths

**Frontend Developer (4-6 months)**
- HTML/CSS/JavaScript fundamentals
- React framework
- API integration
- Deployment

**Backend Developer (4-6 months)**
- Language fundamentals (Python/Node.js/Java)
- Database design
- API development
- Authentication & security

**Full Stack (8-12 months)**
- Frontend skills
- Backend skills
- Database skills
- Deployment skills

**Data Scientist (6-9 months)**
- Python fundamentals
- Statistics & math
- Machine learning
- MLOps & deployment

**DevOps Engineer (5-8 months)**
- Linux fundamentals
- Docker & containers
- Kubernetes orchestration
- Cloud platforms (AWS)
