# /assess - Test Your Knowledge

Assess your knowledge with interactive quizzes and challenges across all 80 roadmaps.

## Usage

```
/assess
/assess react
/assess python level:beginner
/assess system-design level:advanced
/assess [topic] [level]
```

## Assessment Types

### Quick Quiz (5-10 min)
- 5-10 multiple choice questions
- Immediate feedback
- Recommended resources if you score below 70%

### Comprehensive Test (20-30 min)
- 20-30 mixed questions
- Detailed analysis
- Progress tracking
- Skill gap identification

### Practical Challenge (1-2 hours)
- Build a mini-project
- Code review
- Feedback on approach
- Improvement suggestions

## Assessment Levels

**Beginner**
- Basic syntax and concepts
- Simple problem-solving
- Fundamental understanding

**Intermediate**
- Real-world scenarios
- Multiple concepts combined
- Best practices

**Advanced**
- Complex problems
- Edge cases
- Optimization and performance

## Scoring & Feedback

- **0-50%**: Review fundamentals
- **50-70%**: Good progress, some gaps remain
- **70-85%**: Strong understanding, ready to advance
- **85%+**: Expert level, ready for specialized topics

## Next Steps After Assessment

Based on your score:
- **Below 70%**: Review recommended resources
- **70-85%**: Continue to next level
- **85%+**: Ready for advanced topics
