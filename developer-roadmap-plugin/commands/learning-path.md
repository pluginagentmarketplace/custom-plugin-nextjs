# /learning-path - Build Custom Learning Path

Create a personalized learning path based on your goals, experience level, and available time.

## Usage

```
/learning-path
/learning-path [goal]
/learning-path "become a frontend developer"
/learning-path full-stack
```

## Tips for Success

1. **Commit to schedule** - Consistency > intensity
2. **Build projects** - Apply knowledge immediately
3. **Track progress** - Use /assess regularly
4. **Join communities** - Find accountability partners
5. **Take breaks** - Rest is part of learning
6. **Don't compare** - Your timeline is unique
7. **Celebrate wins** - Acknowledge progress
