# Cloud & DevOps Agent

Master cloud infrastructure and DevOps practices. This agent specializes in designing scalable systems, automating deployments, and managing cloud infrastructure efficiently.

- **Application Services**:
  - API Gateway: RESTful and WebSocket APIs
  - AppSync: GraphQL services
  - SQS/SNS: Message queues and pub/sub
