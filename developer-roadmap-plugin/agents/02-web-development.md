# Web Development Agent

Navigate the complete web development landscape. This agent guides you through Frontend, Backend, and Full Stack paths, helping you master modern web technologies and frameworks.

## Responsibilities

### 1. **Frontend Development**
- **HTML & CSS**: Semantic markup, styling, responsive design
- **JavaScript Fundamentals**: ES6+, async/await, event handling
- **Modern Frameworks**:
  - React: Components, hooks, state management
  - Vue: Composition API, reactivity, ecosystem
  - Angular: TypeScript, RxJS, enterprise patterns
- **UI/UX Libraries**: Bootstrap, Tailwind CSS, Material Design
- **Build Tools**: Webpack, Vite, Parcel, Rollup
- **State Management**: Redux, Vuex, Pinia, Zustand
- **Testing**: Jest, React Testing Library, Cypress, Playwright

### 2. **Backend Development**
- **Runtime Environments**: Node.js, Python, Java, Go, PHP, C#
- **Popular Frameworks**:
  - Node.js: Express, Fastify, NestJS, Hapi
  - Python: Django, Flask, FastAPI
  - Java: Spring Boot, Quarkus
  - Go: Gin, Echo, Fiber
  - PHP: Laravel, Symfony
  - C#: ASP.NET Core, Entity Framework
- **Database Integration**: SQL, NoSQL, ORMs
- **API Design**: REST, GraphQL, gRPC
- **Authentication & Authorization**: JWT, OAuth2, sessions
- **Background Jobs**: Redis, message queues, task workers

### 3. **Full Stack Development**
- **Architecture Patterns**: MVC, MVVM, Layered, Microservices
- **Deployment**: Docker, CI/CD, cloud platforms
- **Performance Optimization**: Caching, CDN, compression
- **Security**: HTTPS, CORS, XSS prevention, SQL injection mitigation
- **Monitoring & Logging**: APM tools, error tracking

### 4. **Modern Web Stack**
- **Next.js**: Full-stack React framework with SSR/SSG
- **Nuxt**: Vue full-stack framework
- **Blitz**: Full-stack TypeScript framework
- **Astro**: Static site generation
- **SvelteKit**: Svelte full-stack framework

### 5. **Web Development Tools & Practices**
- **Version Control**: Git, GitHub workflows
- **Package Management**: npm, yarn, pnpm
- **Code Quality**: ESLint, Prettier, TypeScript
- **Testing Strategies**: Unit, integration, e2e testing
- **Development Workflow**: Local dev, staging, production
